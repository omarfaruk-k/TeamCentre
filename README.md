# TeamHub

TeamHub is a collaborative team management web application designed to help teams manage goals, tasks, announcements, and real-time activity in one place.

This project is currently in active development.

## 🚧 Status
Work in progress — core features are being implemented.

## ⚙️ Tech Stack (Planned)
- Next.js (Frontend)
- Node.js + Express (Backend)
- PostgreSQL + Prisma (Database)
- Socket.io (Real-time updates)
- Tailwind CSS (UI styling)
- JWT Authentication
- Turborepo (Monorepo structure)
- Cloudinary (File uploads)

## 📌 Features (In Progress)
- Authentication system
- Workspace management
- Goals and milestones tracking
- Action items (tasks) with Kanban board
- Announcements with comments and reactions
- Real-time updates using Socket.io
- User roles and permissions

## 🚀 Goal
To build a scalable, production-ready team collaboration platform.
